# Python CLI Projects

A collection of small CLI-based Python projects built while learning Python.

## Projects

| Project | Description |
|---|---|
| to do list cli app | A command line app to add, view and delete tasks |
| quiz cli app | A command line quiz game that tests your knowledge |

## How to Run

Clone the repo:
```bash
git clone https://github.com/mvkrishaathithya-web/python---CLI---projects.git
```

Run any project:
```bash
python "to do list cli app.py"
python "quiz cli app.py"
```
